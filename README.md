* 📸 **AI Σάρωση Αναφορών** — Φωτογραφήστε το δελτίο παραγωγής και η εφαρμογή εξάγει αυτόματα όλα τα στοιχεία (Σύνθεση, Υλικά, Ποσότητες, Ημερομηνία) μέσω Google Gemini, OpenAI ή Anthropic Claude.
* 📋 **Διαχείριση & Φιλτράρισμα** — Πλήρες ιστορικό αναφορών με προηγμένη αναζήτηση βάσει σύνθεσης, υλικού και ημερομηνιών σε scrollable πίνακες.

* 📤 **Επαγγελματική Εξαγωγή** — Άμεσος διαμοιρασμός σε PDF (με λογότυπο, γραφήματα & πίνακες) και Excel (με φύλλα σύνοψης & αναλυτικών στοιχείων) μέσω WhatsApp, Email κλπ.
* 🔒 **Ασφάλεια & Offline Λειτουργία** — Τοπική αποθήκευση δεδομένων (SQLite) για πλήρη λειτουργία χωρίς ίντερνετ και βιομετρική αυθεντικοποίηση (FaceID/TouchID) για κλείδωμα της εφαρμογής.

## 🇬🇧 English

### Description
**Synthesis** is a professional mobile application for asphalt plants and production facilities, which fully digitizes production recording through Artificial Intelligence. Built exclusively following Cupertino standards (iOS native look and feel), it delivers a stable, continuous, and premium user experience with complete Dark and Light mode integration.

### Key Features
* 📸 **AI Report Scanning** — Take a photo of the production sheet and the application automatically extracts all data (Composition, Materials, Quantities, Date) via Google Gemini, OpenAI, or Anthropic Claude.
* 📋 **Management & Filtering** — Complete report history with advanced search capabilities by composition, material, and dates displayed in fully scrollable data tables.
* 📊 **Statistics & Graphs** — Real-time production analysis with breakdowns by composition & material, interactive calendar, and beautiful bar charts.
* 📤 **Professional Export** — Instant sharing in PDF (featuring logo, charts & tables) and Excel (with summary & analytical sheets) via WhatsApp, Email, etc.
* 🔒 **Security & Offline Mode** — Secure local data storage (SQLite Repository) ensuring full offline functionality and Biometric Authentication (Biometrics/App Lock) by default.
* 🌍 **Internationalization (9 Languages)** — Built-in multi-language support: *Ελληνικά, English, Deutsch, Français, Español, Italiano, Русский, 中文, हिन्दी.*
