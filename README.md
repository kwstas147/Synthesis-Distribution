Synthesis — Smart Asphalt Production Management
Synthesis is a professional mobile application for asphalt plants and production plants, which fully digitizes production recording through Artificial Intelligence.

Key Features
📸 AI Report Scanning — Take a photo of the production sheet and the application automatically extracts all data (Composition, Materials, Quantities, Date) via Google Gemini, OpenAI or Anthropic Claude.
📋 Management & Filtering — Complete report history with search by composition, material and dates.
📊 Statistics & Graphs — Real-time production analysis with breakdowns by composition & material, interactive calendar and bar charts.
📤 Professional Export — Instant sharing in PDF (with logo, charts & tables) and Excel (with summary & analytical sheets) via WhatsApp, Email etc.
🔒 Security — Local data storage, offline mode and biometric authentication.
🌍 9 Languages ​​— Ελληνικά, English, Deutsch, Français, Español, Italiano, Русский, 中文, हिन्दी.
